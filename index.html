Pengalaman Pembelajaran Jarak Jauh Selama Pandemi Covid-19.

Sejak 2020, saya melakukan pembelajaran jarak jauh karena pandemi covid. Pada awalnya, saya merasa masih bisa mengikuti pelajaran. Awalnya saya juga merasa senang karena bisa belajar dari rumah, tetapi lama kelamaan saya mulai bosan dan kesulitan. Mungkin pada saat itu saya merasa senang karena saya sudah memegang basic pelajaran kelas 10, sehingga ketika pembelajaran online dimulai saya masih bisa mengikuti. Saya juga merasa senang karena saya bisa berada di rumah untuk waktu yang lama. Karena semenjak bersekolah di SMK-SMAK Bogor saya tinggal di kost.

Tetapi ketika kenaikan kelas 11 dan tahun ajaran baru dimulai, saya mulai bosan belajar online. Saya juga kesulitan untuk memahami beberapa materi. Banyak sekali tugas, kuis dan ulangan harian dengan jadwal yang berdekatan. Ditambah lagi saya memiliki 2 adik kecil yang sering berisik dan mengganggu kosentrasi. Saya jadi mudah stres. Saya mudah merasa marah dan sedih. Lalu ketika stres, biasanya saya secepat mungkin mencari teman untuk bercerita. Setelah itu saya akan makan dan tidur. Saya akan mengistirahatkan diri sampai merasa cukup. Karena jika dipaksa, saya akan makin stres. Tetapi, kadang saya terlalu banyak beristirahat (versi halus dari mager). Saya malah menjadi jauh lebih tidak produktif. Menunda tugas sampai h-1 contohnya. Saya lebih memilih nonton k-drama daripada mengerjakan tugas.

Setelah saya menceritakan banyak dampak negatifnya, sekarang saya akan menceritakan dampak positifnya. Saya menjadi lebih melek teknologi. Saya mulai bisa menggunakan canva, membuat dan mengedit video, dan menggunakan aplikasi meeting online seperti zoom dan google meet. Jika tidak pandemi dan tidak belajar online, mungkin saat ini saya tidak akan pernah mencoba melakukan hal-hal itu. Ketika saya menulis ini, pembelajaran online telah dilakukan >1 tahun. Semoga pandemi cepat selesai dan semua dapat kembali normal seperti biasanya!
